Certificate Explorer

Select a company to analyze related certificates. This tool helps you understand how a single company can influence a set of certificates, providing insights into the connections between companies and financial instruments.

How to use this tool

  1. Select a company from the dropdown menu.
  2. The table will update to show all certificates related to the selected company.

When to use this tool

  1. You have multiple certificates in your portfolio, you read/heard a specific news about a company so you want to see how it affects your investments.
  2. You want to invest in a specific company by using certificates.
