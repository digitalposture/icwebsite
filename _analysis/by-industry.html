Certificate Explorer

Select an industry to analyze related certificates. This tool helps you understand how a single industry can influence a set of certificates, providing insights into the connections between industries and financial instruments.

How to use this tool

  1. Select an industry from the dropdown menu or list a set of industries (comma-separated). The list has priority over the dropdown selection.
  2. Click the "Load Data" button.
  3. The sankey graph will update to show the relationships between certificates and the selected industry.
  4. The table will update to show all certificates related to the selected industry.

When to use this tool

  1. You have multiple certificates in your portfolio, you read/heard a specific news about an industry so you want to see how it affects your investments.
  2. You want to invest in a specific industry by using certificates.
