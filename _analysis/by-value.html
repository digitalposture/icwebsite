Certificate Explorer

Select criterias to search for certificates and analyze the results.

How to use this tool

  1. Select a market value criteria.
  2. Select a growth criteria.
  3. Click the search button.
  4. The table will update to show all certificates related to the selected criteria.

When to use this tool

  1. You want to add certificates to your portfolio waiting for the right market conditions.
  2. You want to select certificates based on their market value and coupon rate.
  3. You want to select certificates when they are below their par value and coupon ex-date is close.
