Select a certificate to analyze related underlyings. This tool helps you understand how a single industry can influence a set of certificates, providing insights into the connections between industries and financial instruments.

How to use this tool

  1. Select a certificate from the dropdown menu or paste a list of ISINs. The list has priority over the dropdown selection.
  2. Click the "Load Data" button.
  3. The sankey graph will update to show the relationships between certificates and underlyings.
  4. The table in Underlyings will update to show all underlyings related to the selected certificates.
  5. The table in Certificate Details will update to show further details.

When to use this tool

  1. You have multiple certificates in your portfolio, you want to analyze their underlying assets and exposure.
  2. You want to invest in a specific set of certificates with no overlap.
